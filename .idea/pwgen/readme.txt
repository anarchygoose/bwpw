- This is my first attempt at an actual program to do something useful, so hopefully its 
decent enough to be used in this state while I work on improving it's user friendliness.

- The code makes an ACA Service Desk standard password, two words, two capitals, 4 numbers 
and a special character (exclamation point in this case since it plays nice with pw systems
as many have other special characters blocked).

==============================================================================================

				TO GET LINK ON YOUR DESKTOP

- copy and paste the shortcut to your desktop


==============================================================================================

				HOW TO USE

- clicking on pw generator button will generate passwords

- clicking the clear pws button will clear all the passwords that have been generated 
from the field

- the scrollbar is currently unclickable, so you have to either use the scroll wheel,
arrow keys, or page up / page down to scroll, but I am currently working on figuring
out how to fix this, bare with me plz
